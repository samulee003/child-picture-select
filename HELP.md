# Find My Kid - 帮助中心

## 📖 快速入门

### 第一次使用

1. **准备参考照片**
   - 找 3-5 张孩子的清晰照片
   - 建议包含正面、侧面等不同角度
   - 照片要求：光线充足、面部清晰、无遮挡

2. **加载参考照片**
   - 在"载入围考照片"区域输入照片路径
   - 或直接拖放图片文件到输入框
   - 点击"载入参考照片"按钮

3. **选择照片资料夹**
   - 输入包含班级照或家庭聚会照的文件夹路径
   - 或直接拖放文件夹到输入框
   - 支持子文件夹递归扫描

4. **开始搜索**
   - 调整相似度门槛（建议 0.55-0.65）
   - 点击"开始搜索"按钮
   - 等待 AI 处理完成

5. **查看结果**
   - 结果按相似度从高到低排序
   - 点击"为什么匹配？"查看匹配原因
   - 点击照片可查看大图

## ❓ 常见问题

### 为什么找不到我的孩子？

**可能原因：**

1. **参考照质量不佳**
   - 照片模糊或背光
   - 面部被遮挡
   - 光线太暗或太亮

2. **参考照数量不足**
   - 建议至少 3-5 张
   - 包含不同角度

3. **相似度门槛过高**
   - 尝试降低到 0.5-0.55
   - 然后再逐步提高

4. **照片不是您的孩子**
   - 仔细核对参考照
   - 确认扫描的文件夹正确

### 如何提高匹配准确率？

**最佳实践：**

✅ **参考照选择**
- 使用清晰、正面的照片
- 准备多张不同角度的照片
- 选择光线好的照片
- 避免模糊或遮挡的照片

✅ **参数调整**
- 相似度门槛：0.55-0.65（推荐）
- 显示数量：根据照片总数调整

✅ **环境优化**
- 确保参考照和扫描照片光线条件相近
- 避免参考照是室内、扫描照片是室外的情况

### 支持哪些图片格式？

**支持的格式：**
- JPG/JPEG
- PNG
- GIF
- BMP
- WEBP
- HEIC/HEIF（iPhone 照片）

### 处理速度有多快？

**参考速度：**
- 100 张照片：3-5 秒
- 500 张照片：15-25 秒
- 1000 张照片：30-50 秒

*实际速度取决于电脑配置和照片大小*

### 照片会被上传到云端吗？

**绝对不会！**

- ✓ 所有处理都在您的电脑本地完成
- ✓ 照片 NEVER 上传到互联网
- ✓ 特征向量加密存储
- ✓ 没有追踪、没有广告

## 🔧 高级功能

### 智能增强参考照

如果参考照质量不佳，可以使用智能增强功能：

1. 加载参考照后，点击"✨ 智能增强"按钮
2. 系统会自动优化亮度、对比度和清晰度
3. 可选择使用增强后的照片

### 多角度融合

使用多张参考照可以提高准确率：

1. 加载 3-5 张不同角度的照片
2. 系统会自动融合多个角度的特征
3. 匹配时取最佳匹配分数

### 批量处理

需要处理多组成语？

1. 完成第一次扫描后，结果会自动保存
2. 可以开始新的扫描任务
3. 所有历史记录可在"历史记录"中查看

## 🛠️ 故障排除

### 程序卡顿或崩溃

**解决方案：**
1. 减少同时扫描的照片数量
2. 降低批次大小（设置中调整）
3. 关闭其他占用内存的程序
4. 重启程序

### 匹配结果不准确

**解决方案：**
1. 更换更清晰的参考照
2. 增加参考照数量
3. 降低相似度门槛
4. 确保参考照和扫描照片拍摄时间接近

### 无法加载 HEIC 照片

**解决方案：**
1. 确保系统已安装 HEIF 扩展
2. 或将 HEIC 照片转换为 JPG 格式
3. 使用 iPhone 的"兼容性最佳"设置拍摄

## 🔒 隐私政策

### 数据收集

**我们不收集任何数据：**
- 没有用户账号
- 没有数据上传
- 没有使用追踪
- 没有广告

### 数据存储

**所有数据存储在本地：**
- 参考照特征向量：加密存储
- 扫描历史：本地数据库
- 设置：localStorage

### 数据删除

**您可以随时删除所有数据：**
1. 打开设置 > 隐私设置
2. 点击"删除所有数据"
3. 确认删除
