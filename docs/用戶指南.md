## 快速入門

### 第一次使用

1. **啟動程式**
   - 雙擊程式圖示或運行 `main.py`
   - 等待模型初始化完成（首次使用需要下載模型）

2. **準備參考照**
   - 準備5-15張您小孩的清晰照片
   - 照片中臉部要清晰可見，避免模糊或遮擋
   - 將所有參考照放在同一個資料夾中

3. **開始篩選**
   - 點擊"選擇"按鈕選擇參考照資料夾
   - 點擊"建立參考照庫"按鈕
   - 選擇班級照片資料夾和輸出資料夾
   - 點擊"開始篩選"按鈕

4. **查看結果**
   - 處理完成後，查看輸出資料夾中的照片
   - 查看程式中的結果統計和詳細日誌

## 基本操作

### 參考照設定

1. **選擇參考照資料夾**
   - 點擊"參考照設定"區域的"選擇"按鈕
   - 在彈出的對話框中選擇包含參考照的資料夾
   - 選擇後，資料夾路徑會顯示在輸入框中

2. **預覽參考照**
   - 點擊"預覽參考照"按鈕
   - 使用"上一張"和"下一張"按鈕瀏覽所有參考照
   - 預覽區域會顯示當前照片的文件名、尺寸和大小

3. **建立參考照庫**
   - 點擊"建立參考照庫"按鈕
   - 等待處理完成，系統會顯示找到的參考照數量
   - 如果失敗，請檢查參考照中是否包含清晰的臉部

### 處理設定

1. **選擇班級照片資料夾**
   - 點擊"班級照片資料夾"旁的"選擇"按鈕
   - 選擇包含要篩選的照片的資料夾

2. **選擇輸出資料夾**
   - 點擊"輸出資料夾"旁的"選擇"按鈕
   - 選擇用於存放找到的照片的資料夾

3. **調整相似度閾值**
   - 使用滑塊調整相似度閾值（0.5-0.9）
   - 預設值為0.65，適合大多數情況
   - 較高值（如0.7）減少誤判但可能漏掉一些照片
   - 較低值（如0.6）提高召回率但可能增加誤判

4. **開始處理**
   - 點擊"開始篩選"按鈕開始處理
   - 點擊"停止"按鈕可隨時停止處理
   - 進度條會顯示當前處理進度

### 結果查看

1. **結果統計**
   - 處理完成後，"結果預覽"標籤頁會顯示統計信息
   - 包括找到的照片數量、總處理數量和處理時間

2. **照片列表**
   - 左側列表顯示所有找到的照片
   - 每項包含文件名、相似度分數和信心等級
   - 點擊列表項可在右側預覽照片

3. **照片預覽**
   - 右側預覽區域顯示選中的照片
   - 顯示照片的詳細信息，包括文件名、尺寸和大小

## 高級功能

### 多小孩識別

1. **啟用多小孩模式**
   - 切換到"高級設置"標籤頁
   - 勾選"啟用多小孩識別"選項

2. **添加小孩配置**
   - 點擊"添加小孩"按鈕
   - 在對話框中輸入小孩ID、姓名和參考照資料夾
   - 點擊"確定"保存配置

3. **管理小孩配置**
   - 在列表中選擇小孩配置
   - 使用"編輯"按鈕修改現有配置
   - 使用"刪除"按鈕移除不需要的配置

### 手動標記校正

1. **加載和保存標記**
   - 在"高級設置"標籤頁中
   - 使用"加載標記"按鈕加載之前保存的標記
   - 使用"保存標記"按鈕保存當前標記

2. **標記錯誤識別**
   - 在結果列表中找到錯誤識別的照片
   - 右鍵點擊照片，選擇標記選項
   - 選擇"拒絕"標記為錯誤識別

3. **標記漏掉的照片**
   - 對於系統未識別但實際包含小孩的照片
   - 可以手動添加到結果中
   - 選擇"確認"標記為正確識別

### 性能優化

1. **多線程處理**
   - 在"高級設置"標籤頁中
   - 勾選"啟用多線程處理"選項
   - 調整線程數量（建議為CPU核心數）

2. **圖片質量過濾**
   - 設置圖片質量閾值（0.1-0.9）
   - 預設值為0.3，適合大多數情況
   - 較高值會過濾掉更多低質量照片

3. **緩存管理**
   - 使用"清除緩存"按鈕清理臨時文件
   - 清除緩存後，下次處理會變慢但可能解決一些問題

## 結果解讀

### 相似度分數

- **0.75以上**：高相似度，幾乎可以確定是同一人
- **0.65-0.75**：中等相似度，很可能是同一人
- **0.5-0.65**：低相似度，可能是同一人，但需要人工確認

### 信心等級

- **高**：系統高度確信識別結果正確
- **中**：系統中等確信識別結果正確
- **低**：系統低度確信識別結果正確，建議人工確認

### 統計分析

1. **相似度分布**
   - 顯示不同相似度範圍的照片數量
   - 幫助調整相似度閾值

2. **處理時間**
   - 顯示總處理時間
   - 可用於評估處理效率

3. **質量過濾統計**
   - 顯示因質量過低而被過濾的照片數量
   - 幫助調整質量閾值

## 常見問題

### Q: 程式啟動時提示"模型初始化失敗"

**A:** 這通常是由以下原因造成：
1. 網路連接問題（首次使用需要下載模型）
2. 防火牆阻止模型下載
3. 磁碟空間不足

**解決方法：**
1. 檢查網路連接
2. 暫時禁用防火牆或添加程式到白名單
3. 確保磁碟有足夠空間（至少2GB）
4. 重啟程式重試

### Q: 識別準確率不高

**A:** 識別準確率低可能由以下原因造成：
1. 參考照數量不足或質量不高
2. 參考照中臉部角度、表情、光線變化不大
3. 相似度閾值設置不當

**解決方法：**
1. 增加更多樣化的參考照（建議10-15張）
2. 包含不同角度、表情、光線的照片
3. 調整相似度閾值（降低可提高召回率）
4. 使用手動標記校正錯誤識別

### Q: 處理大量照片時速度慢

**A:** 處理速度慢可能由以下原因造成：
1. 未啟用多線程處理
2. 線程數量設置不當
3. 照片解析度過高
4. 硬碟性能不足

**解決方法：**
1. 啟用多線程處理
2. 調整線程數量（建議為CPU核心數）
3. 設置圖片質量閾值，過濾低質量照片
4. 考慮使用更高性能的存儲設備

### Q: 程式無法讀取某些圖片

**A:** 無法讀取圖片可能由以下原因造成：
1. 圖片格式不受支援
2. 圖片文件損壞
3. 文件路徑包含特殊字符
4. 文件權限問題

**解決方法：**
1. 檢查圖片格式是否受支援（JPG, PNG, BMP, TIFF, WEBP, HEIC, HEIF）
2. 嘗試在其他軟體中打開圖片確認是否損壞
3. 避免使用包含特殊字符的文件路徑
4. 檢查文件權限

## 技巧與建議

### 提高識別準確率

1. **優質參考照**
   - 使用高解析度照片
   - 確保臉部清晰、無遮擋
   - 包含不同角度和表情

2. **合適的相似度閾值**
   - 從預設值0.65開始
   - 根據結果調整：誤判多則提高，漏識多則降低
   - 對於重要照片可使用較高閾值

3. **手動標記校正**
   - 定期檢查識別結果
   - 標記錯誤識別和漏掉的照片
   - 保存標記以提高後續識別準確率

### 提高處理效率

1. **多線程處理**
   - 根據CPU核心數設置線程數
   - 避免設置過多線程（可能導致系統不穩定）

2. **質量過濾**
   - 設置合適的質量閾值
   - 過濾低質量照片可顯著提高處理速度

3. **批量處理**
   - 一次性處理大量照片而非分批處理
   - 避免頻繁啟動和關閉程式

### 結果管理

1. **合理的文件組織**
   - 為不同批次創建不同的輸出資料夾
   - 使用有意義的文件名（如包含日期）

2. **結果備份**
   - 定期備份重要的識別結果
   - 保存手動標記校正文件

3. **結果驗證**
   - 定期抽查識別結果
   - 對於重要照片進行人工確認
