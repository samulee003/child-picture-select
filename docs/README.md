# 小孩照片篩選工具 - 文檔

## 概述

小孩照片篩選工具是一個基於InsightFace臉部識別技術的應用程式，專為從班級照片中精準識別和篩選出特定小孩的照片而設計。本工具採用現代化的GUI界面，支持多小孩識別、手動標記校正、批量處理等高級功能。

## 功能特色

- **高精度臉部識別**：基於InsightFace的buffalo_l模型，提供準確的臉部偵測和特徵提取
- **多小孩識別**：支持同時識別多個小孩，每個小孩可以有獨立的參考照庫
- **手動標記校正**：允許用戶標記錯誤識別的照片，提高後續識別準確性
- **圖片質量評估**：自動評估圖片質量，可過濾低質量照片
- **批量處理**：支持多線程處理，提高大批量照片處理效率
- **實時進度顯示**：詳細的處理進度和預計剩餘時間
- **結果統計分析**：提供相似度分布、信心分布和聚類分析
- **現代化界面**：採用現代設計風格，支持照片預覽和主題切換

## 系統需求

- Windows 10/11 (推薦)
- Python 3.8 或更高版本
- 至少 4GB RAM
- 約 2GB 硬碟空間（包含模型檔案）
- 支援的圖片格式：JPG, JPEG, PNG, BMP, TIFF, WEBP, HEIC, HEIF

## 使用指南

### 基本使用流程

1. **準備參考照**
   - 收集5-15張你小孩的清晰照片
   - 照片中臉部要清晰可見，避免模糊或遮擋
   - 建議包含不同角度、表情、光線的照片
   - 將所有參考照放在同一個資料夾中

2. **建立參考照庫**
   - 點擊"選擇"按鈕選擇參考照資料夾
   - 點擊"建立參考照庫"按鈕
   - 等待模型處理完成

3. **設定處理參數**
   - 選擇班級照片資料夾（要篩選的照片）
   - 選擇輸出資料夾（存放找到的照片）
   - 調整相似度閾值：
     * 0.65: 標準模式（推薦）
     * 0.70: 嚴格模式（減少誤判，可能漏掉一些）
     * 0.60: 寬鬆模式（提高召回率，可能增加誤判）

4. **開始篩選**
   - 點擊"開始篩選"按鈕
   - 等待處理完成
   - 查看結果統計和詳細日誌

### 高級功能

#### 多小孩識別

1. 在"高級設置"標籤頁中啟用"多小孩識別"
2. 點擊"添加小孩"按鈕，為每個小孩創建配置
3. 為每個小孩指定ID、姓名和參考照資料夾
4. 建立參考照庫時，系統會為每個小孩建立獨立的特徵庫

#### 手動標記校正

1. 處理完成後，在"結果預覽"標籤頁中查看結果
2. 對於錯誤識別的照片，可以手動標記為"拒絕"
3. 對於漏掉的照片，可以手動標記為"確認"
4. 使用"保存標記"按鈕保存校正結果
5. 下次處理時，系統會應用這些校正

#### 性能優化

1. 在"高級設置"標籤頁中啟用"多線程處理"
2. 調整線程數量（建議為CPU核心數）
3. 設置圖片質量閾值，過濾低質量照片
4. 使用"清除緩存"按鈕清理臨時文件

## 開發者指南

### 專案結構

```
AI 篩選照片/
├── main.py                 # 主程式入口
├── requirements.txt         # 依賴套件列表
├── src/                   # 源碼目錄
│   ├── core/              # 核心功能模塊
│   │   └── face_recognition_service.py  # 臉部識別服務
│   └── gui/               # GUI模塊
│       └── child_photo_filter_gui.py  # 主界面
├── tests/                 # 測試目錄
│   ├── test_face_recognition_service.py  # 核心服務測試
│   └── test_gui.py        # GUI測試
├── docs/                  # 文檔目錄
├── build_exe.py           # 打包腳本
└── run_tests.py           # 測試運行腳本
```

### 核心API

#### FaceRecognitionService

主要的臉部識別服務類，提供以下方法：

- `initialize_model()`: 初始化InsightFace模型
- `detect_faces(image_path)`: 偵測圖片中的臉部
- `build_reference_gallery(folder, child_profiles)`: 建立參考照庫
- `find_child_in_image(image_path, threshold)`: 在圖片中尋找小孩
- `find_all_children_in_image(image_path, threshold)`: 尋找所有小孩
- `batch_process_folder(input_folder, output_folder, ...)`: 批量處理資料夾
- `add_manual_correction(...)`: 添加手動標記校正
- `cluster_faces(image_paths, ...)`: 對偵測到的臉部進行聚類

#### GUI組件

- `ChildPhotoFilterGUI`: 主GUI類
- `PhotoPreviewFrame`: 照片預覽組件
- `ChildProfileDialog`: 小孩配置對話框
- `ModernTheme`: 現代化主題配置

### 運行測試

1. 運行所有測試：

2. 運行特定模塊測試：

## 常見問題

### Q: 程式啟動時提示"模型初始化失敗"
A: 請確保網路連接正常，首次使用需要下載InsightFace模型。如果問題持續，請檢查防火牆設置。

### Q: 識別準確率不高
A: 嘗試以下方法：
1. 增加更多樣化的參考照（不同角度、表情、光線）
2. 調整相似度閾值（降低可提高召回率）
3. 使用手動標記校正錯誤識別

### Q: 處理大量照片時速度慢
A: 嘗試以下方法：
1. 啟用多線程處理
2. 增加線程數量（不超過CPU核心數）
3. 設置圖片質量閾值，過濾低質量照片
4. 清除緩存後重新處理

### Q: 程式無法讀取某些圖片
A: 請檢查：
1. 圖片格式是否受支援
2. 圖片文件是否損壞
3. 文件路徑是否包含特殊字符
