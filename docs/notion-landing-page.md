# 📥 安裝教學

## 步驟 1：下載

到上方連結，找到 **Find My Kid-x.x.x-Setup.exe** 點擊下載。

## 步驟 2：處理 Windows SmartScreen 警告

> ⚠️ 因為我們是個人開發者，沒有購買微軟的程式碼簽章證書（一年好幾萬元），所以 Windows 會顯示保護警告。**這是正常的，不是病毒！**

當你看到「**Windows 已保護您的電腦**」時：

1. 點擊 **「更多資訊」**（藍色小字，在警告文字下方）
2. 點擊 **「仍要執行」** 按鈕

就這樣！之後就不會再出現了。

## 步驟 3：安裝

1. 選擇安裝位置（建議用預設路徑就好）
2. 點擊「安裝」
3. 安裝完成，桌面會出現 **「大海撈「B」」** 的捷徑

## 步驟 4：啟動

1. 雙擊桌面捷徑
2. 首次啟動需要幾秒鐘載入 AI 模型，請稍等
3. 看到 **「AI 就緒」** 就可以開始了！

# 📖 使用教學

## 準備參考照片

準備 **3~5 張** 小孩的照片：

- 臉部清晰可見，不要太遠或太模糊
- 最好是不同場景 / 角度的照片
- 格式：JPG、PNG、HEIC 都可以

> 💡 參考照片越多、角度越多樣，辨識結果越好。建議 5~10 張。

## 開始搜尋

1. 點擊「**選擇檔案**」或拖放照片 → 載入參考照片
2. 點擊「**選擇資料夾**」 → 選包含大量照片的資料夾
3. 調整門檻值（預設 0.60 通常就很好用）
4. 點擊「**開始搜尋**」
5. 等 AI 分析完成 → 檢視結果
6. 確認後點「**匯出**」→ 選擇存放位置

### 門檻值怎麼調？

| 門檻值 | 效果 |
| --- | --- |
| 0.7 以上 | 非常精準，但可能漏掉一些模糊的照片 |
| 0.55 ~ 0.65 | **推薦** — 精準又不容易漏 |
| 0.4 ~ 0.5 | 找更多，但可能有誤判，建議先複核 |

### 掃描太久？

- 第一次掃描需要分析每張照片，可能要幾分鐘
- 掃描中可以按 **「暫停」** 休息一下，或 **「取消」** 中止
- 之後再掃描同一個資料夾會很快（有快取）

# ❓ 常見問題

**Q: 照片會不會被上傳？**
不會！100% 在你電腦上完成，照片不會離開你的電腦。

**Q: 為什麼安裝時會有警告？**
因為個人開發者沒有花幾萬元買微軟簽章證書。不代表程式有問題，程式碼完全公開。

**Q: 掃描很慢怎麼辦？**
第一次會比較慢（分析每張照片），之後有快取會很快。掃描中可以暫停或取消。

**Q: 找不到小孩的照片？**

1. 增加參考照片數量（5~10 張不同角度）
2. 降低門檻值到 0.4 ~ 0.5
3. 確認參考照片是清晰的正面照

**Q: 有新版本怎麼更新？**
程式內建自動更新！有新版本時會自動跳出通知，一鍵下載安裝。

**Q: 支援 Mac 嗎？**
目前只有 Windows 版。Mac 版開發中。
